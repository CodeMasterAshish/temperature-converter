# 🌡️ Temperature Converter

A simple and beginner-friendly Temperature Converter built using **HTML, CSS, and JavaScript**. This project allows users to convert temperatures between Celsius and Fahrenheit instantly.

## 🚀 Features

* Convert Celsius to Fahrenheit
* Convert Fahrenheit to Celsius
* Simple and responsive user interface
* Input validation for incorrect values
* Clean and easy-to-understand code

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📖 How It Works

### Celsius to Fahrenheit

```
°F = (°C × 9/5) + 32
```

### Fahrenheit to Celsius

```
°C = (°F - 32) × 5/9
```
